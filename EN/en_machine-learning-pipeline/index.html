MACHINE LEARNING PIPELINE

Data Import and manipulation

Importing data Input Stages allow us to use the datasets in the Workspace’s data source. Each type of data source will have its own Input Stage, with the main Input Stages being: • Tabular Input: Input Stage for tabular data type. • Text Input: Input Stage for text data type. • Timeseries Input: Input Stage for sequential data type. • Image Input: Input Stage for image data type. Other types of Input Stage include Image Input for Classification, Image Input for Segmentation.

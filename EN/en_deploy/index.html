Deploy Machine Learning model

Step 1: Run v√† deploy Flow on DSSAMA platform 1. Import data: Choose Input Stage into Dataflow Editor. Create Flows for your problems. 2. Click Run. Once a flow is successfully executed, all the boxes will have green ticks 3. Click Deploy on the toolbar. Step 2: Use Postman deploy Flow Installing Postman After deploying the model successfulling, follow these steps: 1. Call request URL : API /releasedFlows to get the list of deployed flow. 2. Call API /release/ping/<flow_id> to get information of the deployed flow according to <flow_id>. <flow_id> can be retrieved on the address bar of DSSAMA platform 3. Call API /released/runflow/<flow_id> to run deployed flow.

MACHINE LEARNING PIPELINE WITH NLTKSAMA

1. Word Stemming Transformer

  • Step 1: Select Text Input trong Input.Phần parameter:

    • Dataset: here we use BBC_News
    • Selected column: Content
    • Start record: 0
    • End record: 9
    • Click RUN to view input data

  • Step 2: In package nltksama.ml/transform select Word Stemming Transformer.

  • Step 3: Connect Text Input with stage Word Stemming Transformer. The parameters are as follows:

    • language (str) – Select the language of data: 'arabic', 'danish', 'dutch', 'english', 'finnish', 'french', 'german', 'hungarian', 'italian', 'norwegian', 'porter', 'portuguese', 'romanian', 'russian', 'spanish', 'swedish'.
    • ignore_stopwords (bool) – If True, ignore stopwords. Default is False. định.

  • Step 4: Click Run

2. Mwe Tokenization Transformer

  • Step 1: Select Text Input in the Input list.

    • Dataset: BBC NewsBBC_News
    • Selected column: Content
    • Start record: 0
    • End record: 9

  • Step 2: package nltksama.ml/transform select Mwe Tokenization Transformer.

  • Step 3: Nối Text Input(là tập dataset cho bài toán như đã nói ở bên trên là tập BBC_News) vào stage Mwe Tokenization Transformer. The parameters are as follows:

    • mwe (tuple(str) or list(str)): List of phrases to tokenize
    • separator: phrase connector. Default is “-“.

  • Step 4: Click Run

3. Tweet Tokenization Transformer

  • Step 1: Select Text Input in the Input list.

    • Dataset: Ở đây dùng tập BBC_NewsBBC_News
    • elected column: Content
    • Start record: 0
    • End record: 9 Click RUN to view data:

• Step 2: In the package nltksama.ml/transform select Tweet Tokenization Transformer.

• Step 3: Connect Text Input with Tweet Tokenization Transformer stage. For information on the parameters, visit: https://www.nltk.org/api/nltk.tokenize.html?highlight=tweettokenizer#nltk.tokenize.casual.TweetTokenizer

• Step 4: Click Run:

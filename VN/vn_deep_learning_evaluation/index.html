6. TabularData Classification Neural Network

Bước 1: Chọn stage input

  • Chọn Tabular Input trong Input.Phần parameter:
  • Dataset: Ở đây dùng tập Iris dataset (Link dataset: https://www.kaggle.com/arshid/iris-flower-dataset). Tập dataset này được xử lý gồm 6 cột. Cột thứ nhất là index, cột thứ 2,3,4, 5 là các đặc trăng của hoa gồm: Petal Length, Petal Width, Sepal Length, Sepal width. Cột thứ 6 là label gồm 3 lớp tương ứng với 3 loại hoa Iris.
  • Selected column: Petal Length, Petal Width, Sepal Length, Sepal width

Bước 5: Chọn TimeSeries Forecasting Neural Network

Trong package tfsama chọn TimeSeries Forecasting Neural Network bằng cách kích chuột vào nó và sau đó di chuột ra màn hình làm việc, kích chuột vào vị trí cần đặt stage. Trong TimeSeries Forecasting Neural Network người dùng cần tuỳ chỉnh các parameter cho phù hợp với bài toán của mình:

  • Features Column: Chính là các cột vừa được Vectorize trong stage Vectorize Trasform
  • Target Column: Chính là các cột vừa được Vectorize trong stage Vectorize Trasform
  • Layer: dùng để cấu hình Network bằng file Json bằng cách click chuột vào

Ở đây thấy trong Vectorize Trasform ta có 4 feature: Open, High, Low, Close. Đồng thời chọn Window size = 10 còn các tham số khác để mặc định nên ta có config như sau:

  • Window size: tham số này cũng phải đặt tương ứng với Window size trong file config là: 10

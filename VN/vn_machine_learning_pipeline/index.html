MACHINE LEARNING PIPELINE

Scikitsama

Mục đích của hướng dẫn này là để minh họa một số tính năng chính scikit.sama cung cấp.

1. Data Import and manipulation

Từ các tập dataset đã đưa vào Data source, để sử dụng dataset đó trong Flow, người dùng cần import dữ liệu. Để import dữ liệu, người dùng sử dụng các stage Input lựa chọn từ nhóm stage Input. Với mỗi dạng data source sẽ có Stage import dữ liệu riêng , có các stage Input như sau:

  • Tabular Input : Stage input cho dữ liệu nguồn sử dụng là dạng bảng.

  • Text Input : Stage input tương dữ liệu nguồn sử dụng là dạng text.

  • Timeseries Input : Stage input cho dữ liệu nguồn sử dụng là dạng chuỗi thời gian.
  • Image Input : Stage input cho nguồn dữ liệu sử dụng là dạng ảnh.
  • Image Input for Classification :
  • Image Input for Segmentation:
  • Maps Input : Input dữ liệu thể hiện trên bản đồ.
  • Scikit Iput : Input dữ liệu dùng cho thư viện scikit learn, có sẵn một số dataset kinh điển. Không cần phải tạo data source.

Các thao tác

Để import data vào grid flow editor: 1. Click lựa chọn stage Input phù hợp với data source đã tạo và muốn sử dụng( ví dụ đã tạo datasource tabular để sử dụng ta cần chọn Tabular Input ) 1. Click lại vào vùng grid để đưa stage Input đó vào 1. Tùy chọn thuộc tính của stage ở menu phải stage properties 1. Click Run để visualize data đưa vào và quan sát dữ liệu.

Supervised Modeling

1. Classification

Sử dụng các thuật toán Supervised Learning để phân lớp. Trong phần này sẽ sử dụng trực tiếp tập dữ liệu Iris trong phần Input của Scikit.sama để phân lớp. Đây là bộ dataset đã được tiền xử lý gồm 4 features: sepal length (cm), sepal width (cm), petal length (cm) và petal width (cm). Target của bộ dữ liệu là từ 4 features có thể dự đoán được nó thuộc loại hoa Iris nào trong 3 loại hoa đã được encode. - Chọn biếu tượng để chia dữ liệu thành tập train và test. Mặc định là 0.7 cho train và 0.3 cho test: - Ấn Run để xem dữ liệu

NLTK

1. Word Stemming Transformer

  • Bước 1: Chọn Text Input trong Input.Phần parameter:
    • Dataset: Ở đây dùng tập BBC_News
    • Selected column: Content
    • Start record: 0
    • End record: 9 Ấn RUN để xem dữ liệu đầu vào:
  • Bước 2: Trong package nltksama.ml phần transform chọn Word Stemming Transformer bằng cách kích chuột vào nó và sau đó di chuột ra màn hình làm việc, kích chuột vào vị trí cần đặt stage.

  • Bước 3: Nối Text Input(là tập dataset cho bài toán như đã nói ở bên trên là tập BBC_News) vào stage Word Stemming Transformer. Trong Word Stemming Transformer có các tham số để lựa chọn tuỳ thuộc vào từng bài toán:

    • language (str) – Chọn ngôn ngữ cần thực hiện bao gồm: 'arabic', 'danish', 'dutch', 'english', 'finnish', 'french', 'german', 'hungarian', 'italian', 'norwegian', 'porter', 'portuguese', 'romanian', 'russian', 'spanish', 'swedish'.
    • ignore_stopwords (bool) – Nếu được đặt thành True, stopwords không được stemmed and và được trả lại không thay đổi. Đặt False là mặc định.

  • Bước 4: Ấn Run để hiển thị ra kết quả

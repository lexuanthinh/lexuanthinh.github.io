MACHINE LEARING - NLP - PYVISAMA

1. Add Accents Transformer

  • Bước 1: Chọn Text Input trong Input.Phần parameter:
    • Dataset: Ở đây dùng tập VLSP 2016 dataset (Link dataset:). Tập dataset này gồm 3 cột. Cột thứ nhất là index, cột thứ 2 là nội dung đánh giá về sản phẩm của khách hàng, cột thứ 3 là label(đánh giá của khách hàng gồm 3 mức: POS, NEG, NEU)
    • Selected column: sentence

Ấn RUN để xem dữ liệu đầu vào:

  • Bước 2: Trong package pyvisama chọn Remove Accents Transformer bằng cách kích chuột vào nó và sau đó di chuột ra màn hình làm việc, kích chuột vào vị trí cần đặt stage. Bước này dùng để xoá dấu tiếng Việt.

  • Bước 3: Nối Text Input(là tập dataset cho bài toán như đã nói ở bên trên là tập VLSP 2016 dataset) vào stage Remove Accents Transformer.

  • Bước 4: Ấn Run để hiển thị ra kết quả

  • Bước 5: Trong package pyvisama chọn Add Accents Transformer bằng cách kích chuột vào nó và sau đó di chuột ra màn hình làm việc, kích chuột vào vị trí cần đặt stage. Bước này để thêm lại dấu cho tiếng Việt.

  • Bước 6: Nối Remove Accents Transformer vào stage Add Accents Transformer.

  • Bước 7: Ấn Run để hiển thị ra kết quả

Deploy mô hình Machine Learning

Bước 1: Run và deploy Flow trên DSSAMA Platform

  1. Import dữ liệu : Đưa Stage Input vào Dataflow Editor. Tạo các FLow cho bài toán phân tích.
  2. Click Run Stage. Sau khi chạy Flow xong, đảm bảo tất cả các Stage là estimator
  3. Click nút Deploy trên thanh công cụ

Bước 2: Sử dụng Postman để gọi đến Flow đã deploy

Cài đặt Postman

Sau khi deployed thành công, để sử dụng flow đã deployed thực hiện như sau: 1. Call request URL : API /releasedFlows để lấy được danh sách các flows đã deployed. 1. Call API /release/ping/<flow_id> để lấy thông tin mô tả của flow đã deployed theo <flow_id>. <flow_id> cụ thể có thể lấy trên thanh địa chỉ của DSSAMA platform sau khi click nút Deploy

  1. Call API /released/runflow/<flow_id> để run flow đã được deployed. Truyền vào request Request body:
    "datasets": 
    [ 
        { 
            "inputStageId": <lấy từ kết quả của mục 2>, 
            "idCol": <cột id>,
            "labelCol": <cột nhãn>,
            "dataType": <kiểu dữ liệu>,
            "data": <mảng dữ liệu>
        }
    ] 
 }
